# HCCS-DRM 통합 설문

**고교학점제(HCCS)와 일상재구성법(DRM) 통합 설문조사**

📍 **배포**: [https://edu-data.github.io/HCCS-DRM/](https://edu-data.github.io/HCCS-DRM/)

## 설문 구조 (6파트 29문항)

| 파트 | 내용 | 방법 |
|------|------|------|
| **Part 1** | 어제의 하루 재구성 (DRM) | 에피소드 10개 작성 |
| **Part 2** | 에피소드별 심층 진단 | 정보·시간·기회 + 유데모니아 웰빙 |
| **Part 3** | 과목 선택·교육과정·평가 | Q1~Q12 (5점 리커트, 선택형, 개방형) |
| **Part 4** | 종합: 정보·시간·기회 (Iloh 모형) | Q13~Q20 (7점 리커트, 복수 선택, 개방형) |
| **Part 5** | 웰빙(유데모니아)·종합 의견 | Q21~Q29 (5점/7점 리커트, 개방형) |
| **Part 6** | 기본 정보 | 학년, 학교 유형, 지역 |

## 이론적 배경

- **DRM (Day Reconstruction Method)**: Kahneman et al. (2004)
- **Iloh Model**: 교육 기회의 세 축 — 정보(Information), 시간(Time), 기회(Opportunity)
- **Eudaimonia**: 아리스토텔레스의 유데모니아 관점 — 자기성장, 자율성, 몰입, 관계, 의미
